**OpenOxygen** - Next-generation AI Agent platform optimized for Windows native environments.

---

## Features

### Core Intelligence
- **HTN Planner** - Hierarchical Task Network planning
- **MCP Protocol** - Model Context Protocol client
- **OpenAI Protocol** - Tool calling standard implementation
- **AI Cluster** - Multi-model fusion and routing

### Automation Skills (30+)
- **Office** - Word, Excel, PowerPoint, PDF
- **Browser** - Chrome/Edge automation
- **System** - File, clipboard, desktop operations

### Security
- Zero-trust permission system
- Worker Thread sandbox
- AES-256-GCM encryption
- Prompt injection detection

### Performance
- OLB (OxygenLLMBooster) - Rust acceleration
- Flash Attention V3
- TurboKV Cache (3-bit quantization)
