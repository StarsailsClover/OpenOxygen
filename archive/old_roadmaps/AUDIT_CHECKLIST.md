## 四、流程图一致性检查

### 流程图节点对照

```
用户请求 → 读取分层记忆 → 请求格式标准化 → 是否包含文件?
    ↓
是 → 是否为图片文件? → OUV三层融合处理 → 图片语义提取
    ↓
否 → 是否包含文件夹? → 文件安全检查 → OxygenBrowser创建工作区
    ↓
文件语义嵌入 → 启动Ollama → 获取Ollama模型列表
    ↓
模型列表是否一致? → 更新分层记忆 → LLM Router动态选模
    ↓
启动OLB优化推理 → 开LLM判断 → 简单/中等·无需Agent → 开中等LLM直接推理
    ↓
生成回答 → 记忆沉淀-短期 → 输出结果
```
